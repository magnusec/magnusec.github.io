Phishing URL Detection Using Machine Learning Algorithms

Authors: Atharva Kasar, Anjor Rane, Deep Tandel, Sumedha Bhagwat, Jyoti Kundale

Published in: 2025 International Conference on Applications of Machine Intelligence and Data Analytics (ICAMIDA)

This research examines the effectiveness of multiple machine learning algorithms in detecting phishing URLs used in online fraud and credential theft attacks. The study evaluates Decision Tree, Support Vector Machine, Random Forest, Logistic Regression, K-Nearest Neighbours, and Naïve Bayes models across four different datasets.

Experimental analysis achieved detection accuracy of up to 96%, with Random Forest identified as the most effective model. The results demonstrate how machine learning techniques can enhance proactive phishing detection and improve user security in digital environments.

Conference Date: 21–22 August 2025
Conference Location: Aurangabad, India
