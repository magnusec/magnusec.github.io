Peer-reviewed work on phishing URL detection using machine learning

Phishing URL Detection Using Machine Learning Techniques

Published in IEEE in 2025, this paper explores the application of machine learning models to classify URLs as phishing or legitimate using extracted URL features. The research focuses on feature engineering and classifier evaluation strategies with standard supervised learning algorithms.

Such approaches enhance automated URL analysis by leveraging statistical and behavioral features to identify deception patterns and protect users from phishing attacks.
